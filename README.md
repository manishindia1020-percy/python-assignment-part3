# Product Explorer & Logger (Part 3)

This project focuses on file handling, API integration, and exception handling.

## What I implemented

### Task 1 — File Handling

* Wrote notes to a file
* Appended extra lines
* Read file and printed numbered output
* Searched for keywords in file

### Task 2 — API Integration

* Fetched product data from DummyJSON API
* Displayed product details
* Filtered and sorted products
* Retrieved products by category
* Sent POST request (simulated)

### Task 3 — Exception Handling

* Built safe divide function
* Handled file reading errors
* Wrapped API calls with try-except
* Added input validation loop

### Task 4 — Logging

* Created error_log.txt
* Logged errors with timestamps
* Tested with invalid URL and product ID

## How to run

```
python part3_api_files.py
```

## Notes

I tried to handle errors properly so the program doesn’t crash unexpectedly.
